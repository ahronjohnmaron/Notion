Landing Page Lovable Prompt

This prompt is designed specifically for Lovable, which—unlike GPT—is focused on design, layout, and page structure, not copywriting. While GPT prompts help you create the words that sell, Lovable prompts help you bring those words to life visually. The prompt below tells Lovable to build a clean, professional landing page using your exact copy, without rewriting or changing the order of any text. It will automatically handle the layout, spacing, responsiveness, and placeholder visuals, so you end up with a complete, ready-to-preview landing page that reflects your message exactly as written.

You are a professional landing page designer and conversion expert.

Build a clean, responsive, and visually appealing landing page using the *exact* copy I provide below.

Do not rewrite, summarize, or rearrange any text — use it word-for-word and in the same order.

The goal is to create a high-converting **cold traffic event registration page** for a free 2-day virtual workshop.

Branding:

Consider this a branding guide:

[INSERT BRANDING INSTRUCTIONS*]

### Design Instructions:

  • Keep the layout modern, minimal, and mobile-friendly.
  • Use section spacing and hierarchy that clearly separates each part of the page.
  • Make all CTA buttons visually consistent and easy to locate.
  • Add suitable placeholder images (e.g. event hero, host photo, testimonials).
  • Use neutral but professional fonts (e.g. sans-serif) and colors that clients can later customize.
  • Keep button text as provided in the copy (e.g. “Reserve your seat now!”).
  • Do not insert or invent extra wording; only format and style what’s given.
  • Optimize for readability and user flow — headline → benefits → credibility → CTA.

### Structure:

Keep all sections in this exact order:

1. **Hero Section:** Logo (if provided), Headline, Subheadline, Button.

2. **What You Will Learn Section:** Bullet list of topics and learning outcomes.

3. **Workshop Agenda:** Day 1 and Day 2 breakdowns, dates, and times.

4. **What Makes Our Event Different Section:** 3–4 short benefit statements.

5. **Host Bio Section:** Photo, name, short bio paragraph.

6. **Testimonials Section:** Photo/video + quote layout.

7. **Final CTA Section:** Repeat registration button and closing encouragement.
