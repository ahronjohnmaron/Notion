Using the LinkedIn Browser

Use the LI Browser app to log into Ottomatey to log into the LI profiles in your workspace so that you can do any necessary captcha verification, sales nav searching etc.

  • If you have a Mac, download the arm64 version if you have the M1 or M2 chip and the other DMG file if you don’t. If you don’t know which one is right for you, download and install both and one will ultimately fail and suggest the file to be moved to trash; thus, you’ll know it’s the other file. You made need to follow these instructions to open the file after installation.
  • If you have a PC, contact us for the PC version.

Once you have opened the browser, use your Ottomatey credentials to log in. From there, it’ll look like Ottomatey and you can navigate to the profile you want to log into.

Duplicating Campaigns Across Accounts

Go to the campaigns tab and click the duplicate button in the upper right to copy campaigns into other accounts.
