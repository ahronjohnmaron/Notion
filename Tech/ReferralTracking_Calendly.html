Let’s say you have a guest pass page, but want to create tracking links for different sources for traffic so when someone books an interview, you know where the traffic came from. For example, you may have different VAs or affiliates (or even want to know if the traffic came from your IG link in bio versus your email to your list) and you want to track those interviews. Here is the Calendly embed code you’ll want to use.

<div id="calendly-inline-widget" style="min-width:320px;height:630px;"></div>

<script type="text/javascript" src="https://assets.calendly.com/assets/external/widget.js"></script>
<script>

// ?source1=utm_source&source2=utm_medium&source3=utm_campaign&source4=utm_content&source5=utm_term

  var url = new URL(window.location.href);
  var uname =     (url.searchParams.get('uname')) ?   url.searchParams.get('uname') : "";
  var uemail =    (url.searchParams.get('uemail')) ?  url.searchParams.get('uemail') : "";
  var uphone =    (url.searchParams.get('uphone')) ?  url.searchParams.get('uphone') : "";
  var source1 =   (url.searchParams.get('source1')) ? url.searchParams.get('source1') : "";
  var source2 =   (url.searchParams.get('source2')) ? url.searchParams.get('source2') : "";
  var source3 =   (url.searchParams.get('source3')) ? url.searchParams.get('source3') : "";
  var source4 =   (url.searchParams.get('source4')) ? url.searchParams.get('source4') : "";
  var source5 =   (url.searchParams.get('source5')) ? url.searchParams.get('source5') : "";

  Calendly.initInlineWidget({  
  url: 'REPLACETHISWITHYOURCALENDLYBOOKINGLINK?hide_event_type_details=1&hide_gdpr_banner=1',
  parentElement: document.getElementById('calendly-inline-widget'),
  prefill: {
        name: uname,
        email: uemail,
            customAnswers: {
                a1: uphone,
            }
        }, utm: {
            utmSource: source1,
            utmMedium: source2,
            utmCampaign: source3,
            utmContent: source4,
            utmTerm: source5
        }
    });

</script>

However, note the REPLACETHISWITHYOURCALENDLYBOOKINGLINK in the code. Replace that with your Calendly booking page link.

💪🏻 Link Anatomy

Now that you have your webpage set up for tracking, here’s how you build your tracking links. Using our homepage as an example, here is how the core URL looks like:

https://smartbusinessmastermind.com/

In this example, there is zero tracking, other than us knowing that any optins came from the homepage. Where we can start tracking is amending it “source” variables. Now, let’s amend it with one source variable “sbmhome” so that the URL looks like this:

https://smartbusinessmastermind.com/?source1=sbmhome

By amending it with ?source1= and sbmhome, we know for a fact it came in from the SBM homepage (sbmhome is our shortcode for the homepage). For you, you can make it whatever you’d like, but make sure there are no spaces or special characters except for a hyphen or underscore. So you can do something IG-bio or IG_bio but not IG bio. Let’s keep going:

https://smartbusinessmastermind.com/?source1=sbmhome&source2=partners

In this next example, we amended it with &source2= and partners, which then tells us that any prospect who came in via the SBM homepage from one of our partners. But which one?

https://smartbusinessmastermind.com/?source1=sbmhome&source2=partners&source3=bruce_wayne

In this example, we amended it with &source3= and bruce_wayne, which then tells us that any prospect who opts in came in via the SBM homepage from our partner, Batman aka Bruce Wayne. You can keep going following that syntax two more times with source4 and source5.
