Redirect Links

Ok so now that we’ve got the tracking down, you might say that’s one long URL to say or promote. And you’re right! That’s where vanity redirect links come into play.

So for example, if you have smartbusinessmastermind.com as your primary domain where your registration page is hosted, you want another domain separate from that is dedicated for redirects. Part of the reason is because you don’t want to mix up the tech. Let’s say you have WordPress or HighLevel landing page builder on your main domain, if you add the redirect software on that, it could conflict with your pages, blog links, etc. It’s just less messy to have a separate domain.

👥 Our Redirect

Instead of using the long ugly URL, we have the dedicated domain https://smartbusiness.events to use for vanity redirect links. So instead of the long charity URL in the previous example, we can give Bruce a redirect URL like smartbusiness.events/charity and it will redirect to that long URL.

👤 Your Redirect

However, if Bruce wanted to because it’s better for his branding, he can use his own domain like BruceWayne.com and create a redirect like BruceWayne.com/charity and make it redirect to that URL. This means you can do the same if you have a domain you want to use. If you don’t expect creating a lot of redirects, you can use your main domain.

For example, if you have a WordPress site, you can use a plugin like Pretty Links to easily create and manage your links off your main site. Alternately, if you expect you’ll be doing these redirect links a lot, you can get a dedicated domain for creating redirects and use a service like Short.io (what we use to create redirects off smartbusiness.events).

So yes you’ll want to buy a new one, but also check your list of domains you’ve reserved. You might find one there that you haven’t been using that will be great for redirects.
